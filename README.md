# Bored's Board | Online whiteboard

Bored's Board with Next.js and socket.io
## Features

- Drawing lines, circles and rectangles
- Real-time mouse tracking
- Eraser
- Undo/Redo
- Sharing
- Chatting
- Changing backgrounds
- Moving selected area
- Saving canvas
- Placing images
## Made using
- Next.JS
- Socket.IO
- TailwindCSS
- Framer Motion
- Recoil
